Outline of Units and Topics

This course is based on the Ministry of Education's ICS2O curriculum expectations. It is comprised of five units, the first four of which contain four to six activities. Unit 5 is an optional unit for students that complete the first four units and are looking for an extra challenge. The final evaluation is a summative task for the course.

Unit 1: Understanding Computers

Activity 1: Hardware
Activity 2: Software
Activity 3: Operating Systems
Activity 4: Networking
Activity 5: Maintenance & Security
Activity 6: Unit Assignment

Unit 2: Introduction to Python

Task 1: Create a CSCircles account

Task 3: Recommended Python modules

Unit 3: Computers and our Society

Activity 1: Social Impact of Computers

Unit 4: Programming with Python
